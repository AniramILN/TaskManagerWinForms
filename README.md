# TaskManager — Управление задачами

## Возможности
- Добавление, редактирование, удаление задач
- Приоритет (Низкий, Средний, Высокий)
- Дата выполнения
- Фильтрация по статусу
- Сортировка по дате и приоритету
- Цветовая индикация просроченных задач

## Требования
- Windows
- .NET 6/8
- Visual Studio 2022

## Запуск
1. Откройте TaskManager.sln
2. Нажмите F5
